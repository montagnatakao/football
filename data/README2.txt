===============================================
  モンターニャ高尾FC サイト データ更新ガイド
===============================================

このフォルダ（data/）にあるファイルを編集すると
サイトの内容が更新されます。

編集にはメモ帳やテキストエディット等のテキストエディタを使います。
※ Windowsの場合は「メモ帳」、Macの場合は「テキストエディット」
  （テキストエディットは「フォーマット→標準テキストにする」を選んでから編集）


-----------------------------------------------
■ 重要な注意点（必ずお読みください）
-----------------------------------------------

・ファイル名は変更しないでください
・各ファイルの1行目（var ○○ = ...）は変更しないでください
・文字の前後にある " "（ダブルクォーテーション）は消さないでください
・文の中でダブルクォーテーション（"）を使いたい場合は \"  と書いてください
・改行を入れたい場合は  \n  と書いてください（実際の改行ではなく文字として）
・編集後は保存して、ブラウザで index.html を開いて確認してください


===============================================
■ ニュースを追加する（news.js）
===============================================

ファイル: data/news.js

既存の記事をコピーして、データの最後に追加します。

【追加する場所】
最後の } の後ろに , を付けてから新しい記事を追加します。

【例】 ← 最後の記事の後ろに追加する場合：

  },      ← 前の記事の閉じカッコ。カンマ（,）を忘れずに！
  {
    id: 5,
    category: "お知らせ",
    date: "2024.04.01",
    title: "ここにタイトルを書く",
    image: "",
    body: "ここに本文を書く。\n\n改行は \\n を2つ並べると段落が変わります。\n1つだと改行になります。"
  }
];    ← この行は消さないでください

【項目の説明】
  id:       数字。他と重複しない番号にする（5, 6, 7...と増やす）
  category: "お知らせ" / "試合情報" / "レポート" のどれか
  date:     "2024.04.01" のような形式（年.月.日）
  title:    記事のタイトル
  image:    記事の画像パス（なければ "" のまま）
  body:     記事の本文


===============================================
■ 試合を追加する（matches.js）
===============================================

ファイル: data/matches.js

【例】
  { id: 7, type: "リーグ戦", date: "2024-06-01", time: "14:00", home_away: "HOME", team: "モンターニャ高尾", opponent: "相手チーム名", score_home: null, score_away: null, venue: "高尾山口第1G" }

【項目の説明】
  id:         数字（他と重複しない番号）
  type:       "リーグ戦" / "カップ戦" / "練習試合"
  date:       "2024-06-01"（年-月-日）※ニュースと形式が違うので注意
  time:       "14:00"
  home_away:  "HOME" か "AWAY"
  team:       "モンターニャ高尾"（そのまま）
  opponent:   相手チーム名
  score_home: 自チームの得点（試合前は null、試合後は数字で 2 など）
  score_away: 相手の得点  （試合前は null、試合後は数字で 1 など）
  venue:      会場名

【試合結果を入れる場合】
  score_home: null  →  score_home: 2
  score_away: null  →  score_away: 1
  のように null を数字に変更します。


===============================================
■ 選手を追加する
===============================================

2つのファイルを編集する必要があります：

1. data/team.js     ← チーム一覧ページ用
2. data/players.js  ← 選手詳細ページ用

【team.js の例】
  { id: 4, number: 7, position: "MF", name: "佐藤 翔", name_en: "SHO SATO", birthplace: "東京都八王子市", bio: "ドリブルが武器のアタッカー。" }

【players.js の例】
  {
    id: 4,
    number: 7,
    position: "MF",
    name: "佐藤 翔",
    name_en: "SHO SATO",
    birthday: "1999.03.15",
    birthplace: "東京都八王子市",
    height: 170,
    weight: 65,
    features: "スピードのあるドリブルで相手を翻弄するアタッカー。",
    image: ""
  }

  ※ id の数字は両方のファイルで同じにしてください
  ※ 選手写真を使う場合は images/ フォルダに画像を入れて
    image: "images/player_7.jpg" のように指定します


===============================================
■ 静的ページを編集する（pages.js）
===============================================

ファイル: data/pages.js

クラブ概要、フィロソフィーなどのページ内容を編集できます。
content: の中身を書き換えてください。

改行は \n で入れます。


===============================================
■ SNS情報を変更する（social.js）
===============================================

ファイル: data/social.js

YouTube動画を表示する場合：
  youtube_url: "https://www.youtube.com/watch?v=動画ID",

Instagram画像を表示する場合：
  instagram_images: ["images/ig1.jpg", "images/ig2.jpg"]

空にする場合：
  youtube_url: "",
  instagram_images: []


===============================================
■ 選手の写真を追加する
===============================================

1. 写真を images/ フォルダに入れる
   例: images/player_10.jpg

2. data/players.js の該当選手の image を変更する
   image: ""  →  image: "images/player_10.jpg"


===============================================
■ 困ったときは
===============================================

・サイトが表示されなくなった場合
  → 編集した部分で , や " を消してしまっていないか確認してください
  → 分からない場合はバックアップから元に戻してください

・編集前にバックアップを取ることをお勧めします
  → data/ フォルダをコピーしておけば安心です
